# TTSDeckBuilder

## Overview
TTSDeckBuilder is a tool for building and managing decks for Tabletop Simulator (TTS).

## Features
- Create and edit deck configurations
- Export decks to TTS-compatible formats
- Manage card collections
- Preview deck layouts

## Installation
```bash
git clone https://github.com/yourusername/TTSDeckBuilder.git
cd TTSDeckBuilder
```

## Usage
```bash
python main.py
```
